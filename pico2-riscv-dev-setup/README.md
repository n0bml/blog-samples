# Raspberry Pi Pico 2 ARM Development Environment Setup

This directory contains the files mentioned in my blog post: [Raspberry Pi Pico 2 RISC-V Development Environment Setup](https://n0bml.github.io/pico2-riscv-dev-setup/).

- `README.md`: This read me file.
- `install-prerequisites.sh`: Shell script to install the required packages.
- `riscv-setup.sh`: Shell script to compile the RISC-V toolchain.
- `hazard3-setup.sh`: Shell script to compile the Hazard3 variant RISC-V toolchain.
- `multiply.c`: The sample C code.
- `Makefile`: The GNU make file to compile the sample code.
