# Raspberry Pi Pico 2 Basic CMake Project

This directory contains the files mentioned in my blog post: [Raspberry Pi Pico 2 Basic CMake Project](https://n0bml.github.io/pico2-basic-cmake-project/).

- `README.md`: This read me file.
- `CMakeLists.txt`: CMake configuration file to build the Pico 2 executable.
